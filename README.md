# Perspektiven-Museum

Dieses kleine statische Vokabel-Museum unterstützt A2-Lernende dabei, abstrakte Wörter für Begründungen zu üben: Die 20 Karten werden bei jedem Öffnen neu gemischt, zeigen zuerst Artikel und Wort und lassen sich per Tippen umdrehen, damit Erklärung und Beispielsatz sichtbar werden. Die Seite funktioniert ohne Anmeldung, ohne externe Dienste und ohne Tracking; wenn der Browser die Web Speech API unterstützt, kann jede Karte auch auf Deutsch vorgelesen werden.
